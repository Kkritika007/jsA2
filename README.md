# Smoothie Ordering Web Application

A simple web application that allows users to create and order their own custom smoothies. Users can choose different **flavours, bases, and toppings**, and the application automatically calculates the **total price** based on their selections.

## 🥤 Overview

This project was built using **HTML, CSS, and JavaScript** as part of a programming assignment.
The application provides an interactive interface where users can customize their smoothie and see the final price before ordering.

## 🚀 Features

* Choose different **smoothie flavours**
* Select a **base** for the smoothie
* Add multiple **toppings**
* Automatic **price calculation**
* Interactive UI using JavaScript
* Simple and responsive layout

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript**

## 🧑‍💻 How It Works

1. Users select their **smoothie base**.
2. Choose their preferred **flavour**.
3. Add optional **toppings**.
4. The JavaScript logic calculates the **total price** based on the selected options.
5. The final price is displayed to the user.
